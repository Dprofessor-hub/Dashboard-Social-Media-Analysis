# Students Social Media Addiction – Impact Analysis Dashboard
## Project Objective
The objective of this project is to analyze the impact of social media usage on students’ academic performance, mental health, and lifestyle habits. This dashboard aims to uncover patterns of addiction and provide data-driven insights to support better decision-making and awareness.

## Dataset Used
The dataset consists of 705 student records including attributes such as gender, age group, academic level, country, daily social media usage hours, addiction score, sleep duration, mental health score, and academic impact. The dataset is clean with no missing values and is structured for efficient analysis.

## Key Questions (KPIs)
•	What is the average daily social media usage among students?
•	What is the average addiction score?
•	How does social media usage impact academic performance?
•	What is the average sleep duration of students?
•	How is mental health affected by social media usage?
•	Which platforms contribute most to addiction?
•	How does addiction vary across academic levels and demographics?

## Process
1. Data Collection: Gathered structured dataset containing student behavior and performance metrics.
2. Data Cleaning: Ensured dataset had no null or inconsistent values.
3. Data Transformation: Organized data for analysis using Excel.
4. Data Analysis: Used pivot tables to derive insights.
5. Dashboard Creation: Built interactive dashboard using charts, slicers, and KPI cards.
6. Visualization: Designed charts for clear and effective storytelling.

## Project Insights
•	Students spend an average of approximately 4.92 hours daily on social media.
•	Higher addiction levels are associated with lower academic performance.
•	Students with higher usage tend to have reduced sleep duration.
•	Mental health scores show a slight decline with increased addiction levels.
•	Certain platforms show higher addiction scores compared to others.
•	High school students exhibit higher addiction compared to graduates.

## Final Conclusion
The analysis highlights a clear relationship between social media usage and its negative impact on students’ academic performance, sleep, and mental health. The dashboard provides a comprehensive and interactive way to explore these patterns, helping stakeholders understand the importance of managing social media usage effectively.
